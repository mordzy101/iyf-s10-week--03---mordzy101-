# Command Line and Git Practice Project

## Description

This project demonstrates foundational command line skills and Git version control practices. It includes a clean project structure, documented terminal commands, meaningful commit history, a configured `.gitignore`, and a completed branch-and-merge workflow.

## Project Structure

```text
my-project/
├── src/
│   ├── css/
│   ├── js/
│   └── images/
├── docs/
├── tests/
├── README.md
├── .gitignore
└── terminal-log.md
```

## Tools Used

- PowerShell command line
- Git version control
- Markdown documentation

## Skills Demonstrated

- Creating folders and files from the terminal
- Initializing a Git repository
- Staging and committing project changes
- Writing meaningful commit messages
- Creating and merging a feature branch
- Configuring ignored files with `.gitignore`
